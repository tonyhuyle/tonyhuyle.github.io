Galveston Fish Recognition


Fish Recognition Model created with the goal of identifying common species of fish that originate from the Galveston area. The model is trained on 10 different species that include the following:

  • Red Drum
  • Black Drum
  • Crevalle Jack
  • Southern Flounder
  • Speckled Trout
  • Southern Kingfish
  • Atlantic Croaker
  • Sheepshead
  • Ladyfish
  • Gafftopsail Catfish

Model was trained via transfer learning using the MobileNetV2 model as a base and adapting the model specifically for fish classification. The dataset images are preprocessed and normalized before being split into training and testing sets. The model is fine-tuned with data augmentation, drop out layers, batch normalization, and a custom learning rate scheduler. It also uses early stopping and learning rate reduction callbacks for efficient training. It has proven to effectively identify the desired fish species.

HTML, Javascript, Python, Tensorflow, NumPy, Matplotlib, Pandas
